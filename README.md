# gbwork
GB/GBC開発勉強の作業リポジトリ。

## 開発環境
ラズパイ4でDockerイメージの開発環境を使用しています。 \
amd64向けしかないので `QEMU` でエミュレートして動かしてます。

### Dockerfile
[gbdk2020 dcoker image](https://github.com/tsu-kunn/docker_gbdk/tree/master/gbdk2020)
